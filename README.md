# Oceancodes

A modern developer portfolio showcasing my projects, skills, and creative web development work.

## About the Project

Oceancodes is a personal portfolio website designed to present my development projects, technical skills, and experience in an interactive and visually appealing way.

The website features smooth animations, responsive design, and modern UI components.

## 🛠 Technologies Used

This project is built using modern web development tools:

* React
* Vite
* TypeScript
* Tailwind CSS
* shadcn/ui
* Framer Motion

## Project Structure

```
src/
 ├── components   # Reusable UI components
 ├── pages        # Website pages
 ├── assets       # Images and static files
 ├── hooks        # Custom React hooks
 ├── lib          # Utility functions
 └── App.tsx      # Main application component
```

## Running the Project

1. Install dependencies

```
npm install
```

2. Start development server

```
npm run dev
```

3. Open in browser

```
http://localhost:8080
```

## Build for Production

```
npm run build
```
